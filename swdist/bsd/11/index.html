NetBSD 1.1

The FAQ will explain how to install this version.

The boot/copy files

  • RX50-BOO.11A 257k NetBSD 1.1 install RX50 flop 1 van 7
  • RX50-CP1.11A 401k NetBSD 1.1 install RX50 flop 2 van 7
  • RX50-CP2.11A 401k NetBSD 1.1 install RX50 flop 3 van 7
  • RX50-CP3.11A 401k NetBSD 1.1 install RX50 flop 4 van 7
  • RX50-CP4.11A 401k NetBSD 1.1 install RX50 flop 5 van 7
  • RX50-CP5.11A 401k NetBSD 1.1 install RX50 flop 6 van 7
  • RX50-CP6.11A 49k NetBSD 1.1 install RX50 flop 7 van 7
  • TK50FIL1.11A 257k NetBSD 1.1 install TK50 tape file 1 of 2
  • TK50FIL2.11A 2049k NetBSD 1.1 install TK50 tape file 2 of 2

RX50-BOO.11A and TK50FIL1.11A have the program to label the disk and copy the miniroot with the kernel and bootblocks into the swap partition, and the other files are the miniroot.

The install files

  • BIN$TAR.GZ 992k bin.tar.gz
  • DEV$TAR.GZ 9k dev.tar.gz
  • ETC$TAR.GZ 65k etc.tar.gz
  • SBIN$TAR.GZ 1747k sbin.tar.gz
  • USR$BIN$.GZ 8995k usr.bin.tar.gz
  • USR$INCL.GZ 653k usr.include.tar.gz
  • USR$LIB$.GZ 984k usr.lib.tar.gz
  • USR$LIBE.GZ 3205k usr.libexec.tar.gz
  • USR$MISC.GZ 8k usr.misc.tar.gz
  • USR$SBIN.GZ 3593k usr.sbin.tar.gz
  • USR$SHAR.GZ 6318k usr.share.tar.gz
  • VAR$TAR.GZ 31k var.tar.gz

These are supposed to be gzipped tar files, but they might be straight tar files.
