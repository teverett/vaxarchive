Old NetBSD versions

The official NetBSD site now only has 1.3.x, 1.4.x and 1.5.x online. But some VAX aficionados want to play with older versions of NetBSD. So here are 1.0A (incomplete) and 1.1. The other older versions of NetBSD for the VAX are stored at ftp://archives.netbsd.org/pub/NetBSD-archives/

  • NetBSD 1.0A Only boot/copy/miniroot files
  • NetBSD 1.1 Boot/copy files and full distribution
